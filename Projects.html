What is a Sim Racing Wheel?

A sim racing wheel is a specialized input device used in racing simulations to recreate the feel of real-world driving. Unlike standard controllers, sim racing wheels provide steering precision, force feedback, and pedal control, offering a highly immersive experience. They are widely used by gamers, hobbyists, and professional sim racers to simulate realistic driving mechanics.

Custom Sim Racing Setup (Ongoing Project)

I am currently building a custom sim racing setup from scratch to replicate the feel of a real racing vehicle. This project involves a combination of electronics, mechanical components, and custom configurations, requiring both technical skill and hands-on effort.

  • Components Used:
  • Arduino Leonardo – Main controller and USB interface
  • Rotary Encoder – Detects steering wheel rotation accurately
  • Motor Driver – Drives the motor to generate force feedback
  • E-bike Motor (scrap) – Repurposed to simulate steering resistance
  • Accelerator Pedals (scrap) – Used for throttle input

  • Software Used:
  • EMC Utility Lite – Configured to recognize the wheel and pedals as a USB game controller

This is an ongoing project, built with dedication, creativity, and hard work. I continue to improve the force feedback response, pedal mechanics, and overall performance to make the setup more realistic and reliable.

Smart Home Automation Using Arduino and GSM

This project uses an Arduino Uno and a GSM module to control and automate home appliances remotely through a mobile phone. Three bulbs are connected to the system using a relay module and can be individually switched ON or OFF via phone commands. One of the bulbs works automatically using an LDR sensor, which turns the light ON at night and OFF during the day. A power socket is also connected through the relay module and can be controlled remotely. For safety, a gas sensor is integrated into the system. If a gas leak is detected, the GSM module automatically makes a call to the user’s registered mobile number to alert. This project demonstrates smart home control, automation, and safety using Arduino Uno, sensors, and GSM technology.

Smart Home Automation Using Arduino and Bluetooth

This project is built using an Arduino Nano, a Bluetooth module, and a relay module to control home appliances wirelessly. Three light bulbs and one power socket are connected to the relays and can be switched ON or OFF using a mobile phone via a Bluetooth control app. This system provides a simple and efficient way to manage appliances remotely within Bluetooth range, making it suitable for home automation and energy control.
